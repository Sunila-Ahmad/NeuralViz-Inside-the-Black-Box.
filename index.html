Machine Learning, Explained.

How does a machine actually "learn"? It's not magic; it's math visualized. In this sandbox, we will teach a simple Neural Network to recognize a straight line.

The Learning Sandbox

Click "Forward Prop" to send data through the network.

The 4 Pillars of Training

1. Initialization

Weights are set to **random** numbers. The model knows nothing.

2. Prediction

The **Forward Pass** sends data through the neurons to get a result.

3. Error Check

We calculate the **Loss** (how wrong the prediction was).

4. Weight Update

**Backpropagation** uses math to tweak weights and reduce error next time.
